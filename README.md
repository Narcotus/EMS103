# EMS 103 — PWA для скорой помощи Минска

## 🚀 Быстрый старт

### Вариант 1: Запуск локально (простой)
1. Скачайте всю папку `ems-103`
2. Откройте `index.html` в Chrome/Edge
   ⚠️ Для работы модулей нужен локальный сервер:
   - Установите VS Code + расширение "Live Server"
   - ИЛИ запустите `python -m http.server 8000` в папке проекта
   - Откройте `http://localhost:8000`

### Вариант 2: Установка как PWA
1. Откройте в Chrome на телефоне/ПК
2. В адресной строке появится значок "Установить"
3. Нажмите — приложение установится с иконкой

## 📱 Как собрать APK для Android

### Способ A: PWABuilder (самый простой)
1. Разместите проект на бесплатном хостинге:
   - GitHub Pages (бесплатно)
   - Netlify (бесплатно, drag & drop)
   - Vercel (бесплатно)
2. Зайдите на https://www.pwabuilder.com/
3. Введите URL вашего сайта
4. Нажмите "Package for stores" → "Android"
5. Скачайте готовый APK
